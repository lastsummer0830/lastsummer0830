## 👋 About

**Java · Spring 기반 웹 애플리케이션을 백엔드부터 화면까지 만드는 신입 개발자 조아진입니다.**

팀 프로젝트 4개에서 도메인 설계·DB 모델링·협업을 맡았습니다. 개인 프로젝트로는 게임 엔진을 직접 만들며 이어가고 있습니다.
기능을 나열하기보다 _어떤 문제를 왜 그렇게 풀었는지_ 설명할 수 있는 개발을 지향합니다.

**이런 걸 잘합니다**

- 🧩 **얽힌 요구사항을 구조로 정리** — 여러 도메인에 흩어진 상태 변경을 하나의 서비스 계약으로 모으고, 데이터 정합성을 설계 단계에서 잡습니다.
- 🎨 **화면부터 DB까지 혼자 완결** — 디자이너 없이 UI를 설계·구현하고, 백엔드 로직과 쿼리까지 이어 붙입니다.
- 🔍 **AI를 감독·검증하며 활용** — 결과를 그대로 쓰지 않고 근거를 직접 확인해 문서와 코드를 관리합니다.

> 요식업 현장에서 조리 자격 없이 실력만으로 헤드셰프까지 오른 뒤 개발로 전향했습니다.
> 낯선 일을 빠르게 익혀 성과로 만드는 태도를 그대로 코드에 옮기고 있습니다.
